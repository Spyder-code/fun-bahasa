# Fun Bahasa Web Apps


## Post
End point for Post article/naskah

1. `GET` /api/post   > get all post


2. `POST` /api/post  > create a post
   * field: <br>
             `tittle`, `author`, `excerpt`, `content`, `category`, `type`.

3. `GET` /api/post?category={id}
    * field: <br>
    `category` 
4. `GET` /api/post?author={id}
    * field:<br>
    `author id`
